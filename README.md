# powerdns-tools

Bash scripts for working with the PowerDNS API, and for auditing the list of zones.

## Install
```
cd /usr/local/src/
git clone https://git.stack-source.com/msb/powerdns-tools.git
cd powerdns-tools
bash install.sh
```

Edit /usr/local/etc/pdns.conf with your values.
